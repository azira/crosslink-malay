About WIKIM2E

WIKIM2E actually stands for Wikify Malay to English. It is developed for a QUT Project called "Cross Language Wikipedia Link Discovery" with the frame time of five months. It is developed on September 2012 and was extended as a Greasemonkey script where users can install the script in Firefox with Greasemonkey extension.

WIKIM2E anchors words from a Malay Newspaper and directs user to the English Translated Wikipedia Collection. Basically it is a Link-to-wiki online application for those who wants to understand a Malay Word in more detailed information. In this case, Wikipedia provide more richer information in the world wide web.

The website is implemented in Google App Engine while the script is coded in Javascript.

Versions

Currently, WIKIM2E Version 1.0 only wikify two Malay Newspapers:

  • Mediapermata - Brunei Darussalam Online Newspaper
  • Berita Harian - Malaysia Online Newspaper
