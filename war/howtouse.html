How to use

There are two ways you can use WIKIM2E. You can either use the website application (disadvantage: will take a long time to load and you have to copy and paste website address) or download the script and install it on your Firefox Web Browser.

Recommended Web Browser

Morzilla Firefox

How to use the website

Firstly, you need to find a newspaper article website either from Mediapermata or Berita Harian and copy paste the web address in the textbox of the webpage. Then press "Go" button, it will prompt you message saying it will take up to 1 minute to load.

You will find a loaded webpage of the newspaper with the Malay words anchored. It will take awhile for it to display the HTML page.

How to use the script

The Recommended method. The only requirement for this is you need to install Greasemonkey extension and install the WIKIM2E script you downloaded from userscript. Further instruction to install Greasemonkey scripts can be found here.

After installation, you will be able to see Malay words anchored in your Firefox.
